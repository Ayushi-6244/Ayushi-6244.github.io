HR Metrics Analysis Report

I generated a synthetic HR dataset (~5k employees) and analysed attrition, attendance, training and performance. I built KPIs, a lightweight logistic baseline to estimate attrition risk, and charts for leadership reporting.

Sales Performance Dashboard

An interactive analysis over ~10k transactions across four regions. I identified revenue leakage (discounting patterns, low‑margin SKUs), surfaced seasonality and built a dashboard for quick KPI monitoring.

  • Python
  • SQL
  • Power BI/Tableau
  • SQLite

Event Analytics & Process Optimization

I analysed event attendance and feedback to spot engagement bottlenecks and capacity issues. The report tracks utilisation %, satisfaction trends and before/after process changes to improve turnout and throughput.
