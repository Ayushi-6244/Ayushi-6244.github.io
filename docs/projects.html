HR Metrics Analysis Report

Problem: Identify the key drivers of attrition and training outcomes using workforce data.
Approach: Generated a synthetic HR dataset (~5,000 employees), defined HR KPIs (attrition, attendance, training completion, performance), and performed analysis to surface trends and risk patterns. Built a baseline attrition-risk model and visual summaries for reporting.
Output: Stakeholder-ready charts, KPI views, and an HR insights report that supports workforce planning and retention analysis.

  • Role: HR Analyst
  • Role: People Analytics
  • Python
  • Pandas
  • Matplotlib
  • SQL

Sales Performance Dashboard

Problem: Understand revenue leakage and performance drivers across regions and product SKUs.
Approach: Analyzed ~10,000 transactions across four regions to identify discounting patterns, low-margin SKUs, and seasonal trends. Designed KPI views and a dashboard for quick performance monitoring.
Output: Interactive dashboard and insights that highlight where margin and revenue improvements are most likely.

  • Role: BI Analyst
  • Role: Data Analyst
  • Python
  • SQL
  • Power BI/Tableau
  • SQLite
GitHub Repository

Event Analytics & Process Optimization

Problem: Improve operational execution and engagement outcomes for recurring events.
Approach: Analyzed attendance and feedback data to identify engagement bottlenecks, capacity constraints, and process gaps. Tracked utilization and satisfaction trends and compared before/after process changes to evaluate improvements.
Output: A clear operational report with KPIs and recommendations to improve throughput, planning, and participation outcomes.

  • Role: Operations Analyst
  • Role: Business Analyst
  • Python
  • Pandas
  • Matplotlib
  • Process Mapping
